### Color Scheme

The website uses a purple and pink gradient theme. To change colors, modify these CSS variables:

```css
/* Primary gradient colors */
background: linear-gradient(45deg, #8a2be2, #ff1493, #00ff88);

/* Card hover effects */
border-color: #8a2be2;
```
